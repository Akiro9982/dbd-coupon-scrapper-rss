# DBD Coupon Scrapper (RSS)

## ¿Qué es?
- Básicamente, es un pequeño script realizado en el lenguaje de programación Python, el cual se encarga de realizar un proceso de scrapping a la web [DBDCoupons](https://dbdcoupons.com/?) y extrae los códigos de canje más recientes para el videojuego **Dead By Daylight**, y los sirve en un archivo .XML accesible públicamente desde este [enlace](https://akiro9982.github.io/dbd-coupon-scrapper-rss/feed.xml).

## ¿Qué puedo hacer con esto?
- Puedes importar el enlace anterior en algún lector RSS o utilizarlo para recibir notificaciones en un bot de discord **_(Es el uso que le estoy dando actualmente)_** o darle el uso que requieras.
